Завдання на лабораторну роботу №1.

Загальний опис інформаційної системи.
Забезпечення CRUD операцій для інформаційного сайту на задану тематику.

Front-end: Angular
Back-end: Java (Maven, Servlet, Json, OOP, Collections), Tomcat, FS

На сторінці відображаються декілька об'єктів за встановленою темою (назава картинка коротка характеристика). 
Забезпечується виконання CRUD операцій із збереженням інформації у файлі.


Етапи створення:
Система створюється у три етапи, кожен з яких виконується у вигляді лабораторної роботи.

Лабораторна 1. Пілотний проект. 
Інтерфейс користувача (Angular) з отриманням інформації у форматі JSON через Servlet (doGet). 
Приклад відображення одного об'єкту.

1. Створити Eclipse dynamic web project для розробки зазначеної інформаційної системи.
2. В межах зазначеного проекту розробити back-end частину на основі сервлету,
   що обробляє Get запит та видає інформацію щодо одного об'єкту у форматі JSON,
   для чого використати відповідний Entity об'єкт, зокрема метод toString().
3. В межах зазначеного проекту розробити front-end частину з прикладом відображенням одного об'єкту, 
   з отриманням необхідної інформації через Get запит.
4. Інтегрувати розроблені у п.п. 2, 3 частини системи.
5. Створити репозиторій Github для звітів з лабораторних робіт 1-3, 
   та викласти у нього результати виконання п.п.1-4 у гілці lab1.
6. Посилання на репозиторій викласти у таблиці 
   (https://docs.google.com/spreadsheets/d/1G4BHsmXZ-lb5f4sLNCBxrE8AXastAWZrX_qMvI3nB5Y/edit#gid=0).
7. Підготуватися до захисту проекту
