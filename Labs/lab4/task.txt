Завдання на лабораторну роботу №4.

Загальний опис інформаційної системи.
Забезпечення CRUD операцій для інформаційного сайту на задану тематику.

Front-end: Angular
Back-end: Java (Maven, Servlet, Json, OOP, Collections), Tomcat, FS

На сторінці відображаються декілька об'єктів за встановленою темою (назава картинка коротка характеристика). 
Забезпечується виконання CRUD операцій із збереженням інформації у файлі.


Етапи створення:
Система створюється у три етапи, кожен з яких виконується у вигляді лабораторної роботи.

Лабораторна 4. PostgreSQL. 
Інтерфейс користувача (Angular) з виконанням CRUD операцій 
через Servlet із збереженням у базу даних. 

1. На основі гілки git репозиторію 
   з результатами попередньої лабораторної роботи lab3 створити гілку lab4.
2. В межах зазначеного проекту удосконалити back-end частину,
   за рахунок забезпечення виконання усіх CRUD операцій 
   та збереження інформації у базу даних PostgreSQL.
3. Скрипти створення бази даних з назвою student<Number> 
(Number - відповідно до номеру по списку), та таблиці  надати в окремому .sql файлі.
4. Результати викласти у репозиторій GitHub у гілці lab4.
5. Підготуватися до захисту роботи.
