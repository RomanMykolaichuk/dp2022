Опис проекту для розробки в ході лабораторних робіт 1-3.

Забезпечення CRUD операцій для інформаційного сайту на задану тематику.

Frontend: Angular
Back-end: Java (Maven, Servlet, Json, OOP, Collections), Tomcat, FS

На сторінці відображаються декілька об'єктів за встановленою темою (назава картинка коротка характеристика). 
Забезпечується виконання CRUD операцій із збереженням інформації у файлі.


Етапи створення:

Лабораторна 1. Пілотний проект. 
Інтерфейс користувача (Angular) з отриманням інформації у форматі JSON через Servlet (doGet). 
Приклад відображення одного об'єкту.

Лабораторна 2.
Додатково. Відображається UML-діаграма класів. Викоритовуються інтерфейси. Об'єкт зберігається у файлі. 
Можливість редагування об'єкту (doPut).

Лабораторна 3.
Додатково. Відображення усіх об'єктів. Повністю CRUD операції зі списком об'єктів із збереженням у файлі. 


Порядок виконання:

Виконується послідовно з використанням репозиторію GitHub.
Створюється гілка lab1, у якій відпрацьовується лабораторна робота 1.
Далі на базі lab1 створюється гілка lab2, у якій відпрацьовується лабораторна робота 2.
Далі на базі lab2 створюється гілка lab3, у якій відпрацьовується лабораторна робота 3.
